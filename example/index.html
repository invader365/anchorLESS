Using the example

1. Install npm modules


npm install

2. Run npm start


npm start

3. Edit styles.less located on the example folder


cd /example

Your browser will reload after saving changes

Semantic

1. Import anchor.less at the top of your less file: 


// Relative path to anchorless, ie:
@import "../node_modules/anchorless/anchor.less";

2. Write anchorless semantic


#button {
    .gradient.to-bottom(#6e8efb, #a777e3);
    .border(0 1px 2px, solid, #9c33ed);
    .padding(0.2em 0.8em);
    .rounded(4px);
    .width(25%, 640px);
    .shadow.inset(0 5px 15px, #d197fe, .3);
    .color(@white);
    .size(128px, 36px);
    .font(bold 12px @d 34px, 'Arial');
}

Compiles to:


#button {
    background: #6e8efb none repeat-x center;
    background-image: -webkit-linear-gradient(180deg, #6e8efb, #a777e3);
    background-image: -moz-linear-gradient(180deg, #6e8efb, #a777e3);
    background-image: linear-gradient(180deg, #6e8efb, #a777e3);
    border-width: 0 1px 2px;
    border-style: solid;
    border-color: #9c33ed;
    padding: 0.2em 0.8em;
    -webkit-border-radius: 4px;
    -moz-border-radius: 4px;
    border-radius: 4px;
    -webkit-box-shadow: 0 5px 15px rgba(0, 0, 0, 0.3) inset;
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.3) inset;
    color: #ffffff;
    width: 128px;
    height: 36px;
    font: bold 12px / 34px 'Arial';
}


@media all and (min-width: 640px) {
    #button {
        width: 25%;
    }
}

3. Try the less live-compiler to see how it works!
