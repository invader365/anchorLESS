Usage

1. Download the "anchor.less" file

2. Reference for use globally with: 

@import "anchorless";

3. Use your favorite less compiler or try Prepros (it's FREE and open-source)

Example:

#button {
   .gradient.to-bottom(#a53df6, #7c16cb);
   .border(0 1px 2px, solid, #9c33ed);
   .rounded(7px);
   .width(25%, 640px);
   .shadow.inset(0 1px 0 0, #d197fe, .9);
   .color(@yellow);
   .size(56px, 128px);
   .font(bold 12px @d 56px, 'Arial');
 }

Compiles to:

#button {
     background: #a53df6 none repeat-x center;
     background-image: -webkit-linear-gradient(180deg, #a53df6, #7c16cb);
     background-image: -moz-linear-gradient(180deg, #a53df6, #7c16cb);
     background-image: linear-gradient(180deg, #a53df6, #7c16cb);
     border-width: 0 1px 2px;
     border-style: solid;
     border-color: #9c33ed;
     -webkit-border-radius: 7px;
     -moz-border-radius: 7px;
     border-radius: 7px;
     -webkit-box-shadow: 0 1px 0 0 rgba(208, 149, 254, 0.9) inset;
     box-shadow: 0 1px 0 0 rgba(208, 149, 254, 0.9) inset;
     color: #ffff00;
     width: 56px;
     height: 128px;
     font: bold 12px / 56px 'Arial';
 }@media all and (min-width: 640px) {
     #button {
         width: 25%;
     }
 }
