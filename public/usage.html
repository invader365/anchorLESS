Usage

1. Download anchor.less

2. Reference for use globally with: 

@import "../node_modules/anchorless/anchor.less";

Example:

button {
     .gradient(to-bottom, #a53df6, #7c16cb);
     .border(0 1px 2px, solid, #9c33ed);
     .rounded(7px);
     .shadow(0 1px 0 0, #d197fe, .9, inset);
     .color(yellow);
     .size(56px, 128px);
     .font(12px, 56px, bold, 'Arial');
     .breakpoint({
          .size(25%, _);
      }, 640px);
 }

Compiles to:

button {
      background-image: -webkit-linear-gradient(180deg, #a53df6, #7c16cb);
      background-image: -moz-linear-gradient(180deg, #a53df6, #7c16cb);
      background-image: linear-gradient(180deg, #a53df6, #7c16cb);
      border-radius: 7px;
      -webkit-box-shadow: 0 1px 0 0 rgba(209, 151, 254, 0.9) inset;
      box-shadow: 0 1px 0 0 rgba(209, 151, 254, 0.9) inset;
      color: yellow;
      width: 56px;
      height: 128px;
      font-size: 12px;
      line-height: 56px;
      font-weight: bold;
      font-style: 'Arial';
  }@media all and (min-width: 640px) {
     button {
          width: 25%;
      }
  }
