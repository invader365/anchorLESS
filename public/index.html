Anchor Less is a collection of less Mixins, Assets and Utilities. These enables you to create blocks of reusable css, avoiding duplication of code and making it more maintenable.

Features

  • The most extensive Less mixins library.
  • Ready to use with html5 frameworks.
  • Faster performance (Less.js powered).
  • Responsive mixins (mobile-first)
  • A single 'anchor.less' file, no dependencies.

Less.js

Anchor uses Less.js, a javascript pre-processor that extends CSS with features like variables, functions and operations.

Supports Less 1.7.1 +

Mixins

  • .back (color, image, repeat, position, query);
  • .border (width, style, color, query);
  • .color (color, back-color);
  • .font (font, family);
  • .gradient (start, stop, degrees, image);
  • .image (url);
  • .opacity (opacity, query);
  • .pseudo (content, width, height)
  • .rounded (radius);
  • .shadow (spread, color, alpha, offset);
  • .size (width, height, query);
  • .transition (property, duration, function, delay);
